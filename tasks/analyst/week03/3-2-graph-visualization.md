# 学習目標
- SQL クエリ結果を効果的にグラフで表現できる
- 複数のグラフタイプ（棒・折れ線・円）を使い分けられる
- 可視化の設定をカスタマイズして見やすいグラフを作成できる

# 作業内容
- 棒グラフの作成と設定（カテゴリー軸、値軸）
- 折れ線グラフの作成と設定（時系列データの表現）
- 円グラフの作成と設定（構成比の表現）
- 各グラフのカスタマイズ（色、ラベル、範囲）
- 可視化の保存と再利用

# 完了条件
- 適切なグラフを選択できる
- 複数のグラフタイプを実装できる

# 成果物
- 保存された可視化（複数のグラフ）
